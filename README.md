# Sikkim Tourism Website

AI Powered Tourism Planning System for Sikkim.

## Project Description
This project helps tourists explore Sikkim by providing:
- Tourist place information
- Smart itinerary planning
- AI chatbot assistance
- Location based recommendations

## Tech Stack
- Python
- Django
- HTML
- CSS
- JavaScript
- SQLite
- GIS Integration

## Setup Instructions

1. Clone the repository

git clone https://github.com/Harshiya-Sameera/sikkim-tourism-website.git

2. Create virtual environment

python -m venv venv

3. Activate environment

Windows:
venv\Scripts\activate

4. Install requirements

pip install -r requirements.txt

5. Run server

python manage.py runserver
